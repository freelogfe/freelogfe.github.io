FreelogApp.QI.resolvePresentableDataUrl(presentableId)

参数

Object object

属性 类型 必填 说明

返回 - promise

示例

// 获取子资源的url
const url = FreelogApp.QI.resolvePresentableDataUrl('1450feb3e49e365d35fe2486f92ddc5a')
