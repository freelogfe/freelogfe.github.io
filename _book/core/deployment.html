组件发布

构建

对'src/app/index.js'进行webpack构建，打包生成一个js文件；

在项目根目录下执行

$ npm run build

发布

在项目根目录下执行

$ freelog-cli publish

根据提示进行输入。账户第一次输入后会保存登录态，过期后需重新登录。
